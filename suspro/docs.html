Примеры анимации цифр

Пример анимации чисел на сайте. Для примера анимируем число 15.

Для использования анимации, нужно число обернуть в любой html-тег и задать data-атрибут data-counter="XXX", где XXX - это число, которое нужно анимировать.

Так же можно задать data-атрибут data-duration="3000" с указанием времени анимации в мс. По умолчанию значение задано 2000 мс.

Еще есть возможно задать повторную анимацию числа, после того как пользователь еще раз при скроле сайта увидит это же число. Для этого нужно использовать data-атрибут data-repeat="true". По умолчанию значение задано false (откл).

Примеры анимации заголовков и текста

Пример анимации залоголовка и текста на сайте. Для примера анимируем заголовок и ниже простой текст.

Для использования анимации заголовка, нужно его обернуть в любой html-тег и задать тегу класс .head-animate.

Еще есть возможно задать повторную анимацию заголовка, после того как пользователь еще раз при скроле сайта его снова. Для этого нужно использовать data-атрибут data-repeat="true". По умолчанию значение задано false, что означает что повтор аниамции отключен.

Так же есть еще параметр data-letter-delay="0.2", который отвечает за задержкe между буквами (по умолчанию 0.08s).

И есть второй параметр data-word-delay="0.4", который отвечает за задержкe между словами (по умолчанию 0.4s).

Чем значение в секундах выставлено больше, тем медленнее анимация.



Для использования анимации текста, нужно его обернуть в любой html-тег и задать тегу класс .text-animate.

To use: in sass - hotkeys

  • pc => notebook (1023.98p)
  • tablet => tablet (991.98px)
  • mob => phone (767.98px)
  • mobMid => phone mid (575.98px)
  • mobSm => phone small (479.98px)
  • mma => media max
  • mmi => media min

  • picture => tag picture with srcset
  • img => tag img in wrap with class _ibg
  • img2x => tag img for 2xRETINA in wrap
  • swiper => slider on swiper
  • menusoc => socials menu
  • check => checkbox
  • sec => section with class and container
  • d => tag div with class

  • vbg => video on background

  • psaimg => position-absolute-img
